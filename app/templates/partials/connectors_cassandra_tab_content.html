CLI (kept as-is)

You can also run the Cassandra connector manually from terminal.

python -m app.connectors.cassandra \
  --contact-points 10.0.0.10,10.0.0.11 \
  --port 9042 \
  --username cassandra \
  --password '<password>' \
  --use-tls \
  --insecure \
  --asset-type OTHER \
  --include-cluster-name-tag \
  --mode dry-run \
  --db-path ./ipocket.db

For full mapping/options, see /docs/cassandra-connector.md.
