CLI (kept as-is)

You can also run the Ceph connector manually from terminal.

python -m app.connectors.ceph \
  --ceph-url https://ceph-mgr.example.local:8443 \
  --username admin \
  --password '<password>' \
  --insecure \
  --asset-type OTHER \
  --include-label-tags \
  --mode dry-run \
  --db-path ./ipocket.db

For full mapping/options, see /docs/ceph-connector.md.
