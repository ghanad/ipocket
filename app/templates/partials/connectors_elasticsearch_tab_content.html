CLI (kept as-is)

You can also run the Elasticsearch connector manually from terminal.

python -m app.connectors.elasticsearch \
  --elasticsearch-url https://127.0.0.1:9200 \
  --api-key '<base64-or-id:key>' \
  --asset-type OTHER \
  --mode dry-run \
  --db-path ./ipocket.db

For full mapping/options, see /docs/elasticsearch-connector.md.
