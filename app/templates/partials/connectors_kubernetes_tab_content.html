CLI

You can also run the Kubernetes connector manually from terminal.

python -m app.connectors.kubernetes \
  --api-url https://kubernetes.example.local:6443 \
  --token '<service-account-token>' \
  --asset-type OS \
  --include-label-tags \
  --mode dry-run \
  --db-path ./ipocket.db

For full mapping/options, see /docs/kubernetes-connector.md.
