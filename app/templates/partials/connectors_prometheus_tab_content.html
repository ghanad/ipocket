CLI (kept as-is)

You can also run the Prometheus connector manually from terminal.

python -m app.connectors.prometheus \
  --prometheus-url http://127.0.0.1:9090 \
  --query 'up{job="node"}' \
  --ip-label instance \
  --mode dry-run \
  --ipocket-url http://127.0.0.1:8000 \
  --ipocket-token '<token>'

For full mapping/options, see /docs/prometheus-connector.md.
